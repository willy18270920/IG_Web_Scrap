此專案為使用Selenium自動化對網紅的Instagram平台進行網路爬蟲
其中皆有兩行是要自行輸入登入帳號密碼
執行順序：先執行"IG_scrap_csv"獲取貼文連結，再執行"IG_post_scrap_csv"獲取貼文資訊

This project involves using Selenium to automate web scraping on Instagram for influencers. 
There are two lines in the code where you need to input your login credentials manually. 
The execution sequence is as follows: first, run "IG_scrap_csv" to obtain post links, then run "IG_post_scrap_csv" to retrieve post information.
